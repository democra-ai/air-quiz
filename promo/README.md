# AIR — promo video (`air-quiz/promo`)

A bilingual **16:9** social promo for **AIR · The AI-Resistance Career Personality Test**
(air.democra.ai). Lives inside the AIR project (this folder), built with **Remotion**.

## Approach
Script + storyboard are produced by a research-backed Workflow (`workflows/script-storyboard.mjs`,
mirroring `ai-video-studio`'s `research-and-script.mjs`): research → 3 drafts → judge → synthesize →
humanize. Winning angle: **curiosity / identity-badge** ("which one are YOU?"). Lead with the MBTI
analogy — *"You know your MBTI. What's your AI-resistance type? 16 career archetypes."* Real live-site
footage (Playwright @2×) with a **Screen-Studio click focus-zoom** on the quiz; a **two-row floating
archetype wall** (homepage feel) for the grid; clean motion-graphics for the axes.

7 beats: `hook (MBTI, hero) · grid (16 types — two floating rows + giant "16") · axes (4 questions, 2 fall to AI / 2 you hold) · quiz (free, ~1 min) · example (researcher → ESRP, ~29%, the rest is yours) · which-one-are-YOU (your slot empty) · CTA (air.democra.ai + share nudge)`.
Framed throughout as **AI resistance** — *how much of your job AI just can't touch* — not replacement.
Captions sit on a soft translucent-white lower-third band (thin serif).

## Audio pipeline (the `ai-video-studio` method — `prepare.py` ported to `scripts/prepare.mjs`)
The canonical script is **`src/script.json`** (per-scene `narration` + `caption` + `visualType`, plus a
`voice`/`rate` per language). `scripts/prepare.mjs` synthesizes each scene's narration with **edge-tts**,
measures real durations with **ffprobe**, and emits **`src/scene-data.ts`** (timed scenes) +
`public/audio/<lang>/scene-NN.mp3`. Nothing is hand-timed.
- **Voiceover** — edge-tts neural female voices: **Ava Multilingual** (EN, +9%) / **晓晓 Xiaoxiao** (ZH, +13%).
- **BGM** — **"Carefree" — Kevin MacLeod** (incompetech.com), **CC BY 4.0**, via `scripts/fetch_bgm.sh`. Credit when publishing:
  *"Carefree" Kevin MacLeod (incompetech.com) — Licensed under CC BY 4.0.*

## Outputs (`out/`, git-ignored — local)
`air-promo-en.mp4` (~70s), `air-promo-zh.mp4` (~75s) — 1920×1080 — + `poster-en.png` / `poster-zh.png` (the floating 16-type wall).

## Rebuild
```bash
npm install && npx playwright install chromium
# 1) capture real site states (researcher → ESRP share URLs from ../scripts/_make-researcher-url.mjs)
RESULT_EN="<hero/grid/quiz urls>" node scripts/capture.mjs
RESULT_EN="<esrp-url-en>" RESULT_ZH="<esrp-url-zh>" node scripts/capture-result.mjs
# 2) audio + timing (edit src/script.json first if changing copy)
node scripts/prepare.mjs                 # edge-tts VO + ffprobe → src/scene-data.ts + audio
bash scripts/fetch_bgm.sh                # Carefree → public/audio/bgm.mp3
# 3) render
npx remotion render AirPromoEN out/air-promo-en.mp4
npx remotion render AirPromoZH out/air-promo-zh.mp4
npx remotion still  AirPromoEN out/poster-en.png --frame=400
```
Edit copy in `src/script.json`, then re-run `prepare.mjs` — timing regenerates automatically.
Generated assets (`public/audio`, `public/shots`, `out`, `src/scene-data.ts`) are regenerated by the scripts.
