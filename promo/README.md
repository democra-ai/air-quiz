# AIR — promo video (`air-quiz/promo`)

A bilingual **16:9** social promo for **AIR · The AI-Resistance Career Personality Test**
(air.democra.ai). Lives inside the AIR project (this folder), built with **Remotion**.

## Approach
Scenes use **real screenshots of the live site** (captured with Playwright at 2×) animated with
Ken-Burns zoom + an on-screen cursor — so it reads like a real product demo — interleaved with a
few clean motion-graphic beats. Opening = the real Hero (doubles as the poster/cover).

8 beats: `s1 hero/poster · s2 archetypes · s3 four-axes · s4 quiz (cursor+zoom) · s5 example (Software Developer→ESFP) · s6 result reveal · s7 macro · s8 CTA`.

## Audio (same engine as `democra-ai/ai-video-studio`)
- **Voiceover** — edge-tts neural female voices: **Ava Multilingual** (EN) / **晓晓 Xiaoxiao** (ZH). (`scripts/gen-vo.sh`)
- **BGM** — **"Carefree" — Kevin MacLeod** (incompetech.com), **CC BY 4.0**. When publishing, credit:
  *"Carefree" Kevin MacLeod (incompetech.com) — Licensed under CC BY 4.0.*

## Outputs (`out/`, git-ignored — local)
`air-promo-en.mp4`, `air-promo-zh.mp4` (1920×1080, ~54s) + `poster-en.png` / `poster-zh.png`.

## Rebuild
```bash
npm install
npx playwright install chromium
# capture real site states (needs the ESFP result share URLs)
RESULT_EN="<share-url-en>" RESULT_ZH="<share-url-zh>" node scripts/capture.mjs
bash scripts/gen-vo.sh                  # edge-tts voiceover + durations.json
python3 scripts/gen-bgm.py              # (or fetch Carefree) → public/audio/bgm.mp3
npx remotion render AirPromoEN out/air-promo-en.mp4
npx remotion render AirPromoZH out/air-promo-zh.mp4
npx remotion still  AirPromoEN out/poster-en.png --frame=30
npx remotion studio                     # live preview
```
Scene timing (`src/scenes.ts`) is derived from the measured voiceover durations.
Generated assets (`public/audio`, `public/shots`, `out`) are git-ignored and regenerated by the scripts.
